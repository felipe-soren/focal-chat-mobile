# Focal Chat Mobile

This project is a mobile client for Focal chat.

## Requirements

For development, you will need Node.js and React Native installed.

## Start local project on Android Platform

    $ cd focal-chat-mobile/
    $ yarn install || npm -i
    $ react-native run-android
    $ node index
